# 🎯 Competitor Intelligence Dashboard - Complete Project Overview

## 📋 Project Summary

This is a **production-ready, AI-powered Competitor Intelligence Dashboard** that automates competitor analysis using a multi-agent system. It scrapes competitor websites, extracts business intelligence, generates strategic insights, and displays everything in a modern, professional dashboard.

## 🔄 Data Flow

```
1. USER INPUT (URLs or Location + Company Type)
                ↓
2. COMPANY DISCOVERY (Find companies)
   - DuckDuckGo/Google/Bing search
   - URL filtering & validation
   - Company name extraction
                ↓
3. SCRAPER AGENT (Collect data)
   - HTTP requests
   - HTML parsing
   - Extract: Services, Tech, Clients, Projects
                ↓
4. CLEANER AGENT (Structure data)
   - Data validation
   - Deduplication
   - Format standardization
                ↓
5. ANALYZER AGENT (Analyze data)
   - Service frequency
   - Technology patterns
   - Trend identification
                ↓
6. COMPARISON AGENT (Compare - V2)
   - Target vs competitors
   - Gap analysis
   - Competitive positioning
                ↓
7. DATABASE (Store data)
   - Companies
   - Services, Tech, Clients, Projects
   - Reports, Comparisons
                ↓
8. DASHBOARD (Display results)
   - 15+ dynamic sections
   - Charts & visualizations
   - Strategic insights
```

```bash

PORT=3001 npm start
```

## Complete Start Script (Runs both in background)

```bash
cd ~/competitor-intelligence-dashboard

# Kill existing processes
sudo lsof -ti:8000 | xargs kill -9 2>/dev/null
sudo lsof -ti:3001 | xargs kill -9 2>/dev/null
pkill -f uvicorn 2>/dev/null
pkill -f react-scripts 2>/dev/null

# Start backend
cd backend
source venv/bin/activate
python3 -m uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload &
cd ..

# Wait for backend
sleep 5

# Start frontend
cd frontend
PORT=3001 npm start &
cd ..

echo ""
echo "✅ System Ready!"
echo "📊 Frontend: http://localhost:3001"
echo "🔗 Backend: http://localhost:8000"
```
